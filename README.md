# Akena RMMZ Plugins

RPG Maker MZ 전용으로 제작한 Akena 플러그인 모음입니다.
원하는 플러그인을 다운로드하여 `js/plugins` 폴더에 넣고 사용하세요.

---

## 플러그인 목록

| 플러그인 | 버전 | 설명 |
|----------|------|------|
| [Akena_Debug.js](Akena_Debug.js) | v1.00 | 디버그 뷰어 — 파티 HP/MP, 변수, 스위치 실시간 확인 (F10 토글) |
| [Akena_EventHelper.js](Akena_EventHelper.js) | v1.00 | 다수의 스위치/변수 조건을 한 번에 평가 |
| [Akena_Message_ImageShow.js](Akena_Message_ImageShow.js) | v1.0 | 메시지와 함께 이미지 표시 (L/C/R 배치, 페이드, 화자 강조 지원) |
| [Akena_BattleHitBalloon.js](Akena_BattleHitBalloon.js) | v1.0 | 전투 중 피격 시 액터 위에 말풍선 이펙트 표시 |

---

## 다운로드 방법

1. 위 목록에서 원하는 플러그인 이름 클릭
2. 우측 상단 **Download raw file** 버튼 클릭
3. 다운로드된 `.js` 파일을 프로젝트의 `js/plugins` 폴더에 복사
4. RPG Maker MZ 에디터 → 플러그인 관리에서 추가 후 사용

---

## 주의사항

- 모든 플러그인은 **RPG Maker MZ 전용**입니다. MV에서는 동작하지 않습니다.
- 플러그인 적용 전 프로젝트 백업을 권장합니다.
